My Projects

Mental Health website

During the CodeFury 6.0 Hackathon, our team developed a responsive mental health website using HTML, CSS, and JavaScript. The site was designed to address issues like information overload and social media pressure, featuring interactive elements such as self-assessment tools, a chatbot for instant support, mood trackers, a resource library, guided yoga sessions for stress relief, and a call conference feature for live support. We ensured the website was user-friendly and optimized for performance on both desktop and mobile devices. The project was recognized for its innovative approach and potential impact on users managing mental health challenges.

Blind Person To Access Atm

During the Smart India Hackathon, our team created an accessible ATM interface designed for visually impaired users. Using Python, we incorporated features such as one-time password generation and facial recognition to ensure secure and seamless authentication. The interface was designed to be user-friendly, allowing visually impaired individuals to navigate and perform banking transactions independently. Our solution aimed to enhance both accessibility and security, providing a reliable and efficient banking experience tailored to the needs of users with visual impairments.

E-Commerce Website

Our team developed a comprehensive e-commerce platform using the MERN stack (MongoDB, Express.js, React.js, Node.js). The platform featured secure payment integration and JWT authentication to ensure transaction security and user data protection. We optimized performance, achieving a 40% reduction in page load times, which greatly improved the user experience. Additionally, our enhancements led to a 30% increase in user engagement, demonstrating the platform's effectiveness in attracting and retaining customers. This project highlighted our ability to build a scalable, secure, and efficient e-commerce solution.
